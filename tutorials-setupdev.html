Forge Tutorial

Getting Started

Required Resources

First of all you need an IDE. In this tutorial I will be showing you how to setup Intellij IDEA, as it is a well supported IDE on Forge's behalf. Once you have an IDE, to setup your development environment you will need to get a copy of the Forge MDK.

Intellij IDEA

To follow along with this tutorial series, you should get a copy of Intellij IDEA. Before you click the link below, the Ultimate Edition costs you money unless you qualify for a free lisence.


The Community Edition provides all you will need to follow along.

Get Intellij IDEA here
Forge MDK

The Forge MDK (Mod Development Kit) is the tool you will need to add blocks, items and more to the game. It is an open-sourced project, currently managed by LexManos.


ALWAYS try to use the latest version of forge!

Get Forge MDK here Support the Forge team here

Creating your Intellij Project

To setup your project properly, you will want to extract the MDK zip into a folder, and rename it to the name of your project (You can change this later if you change your mind about the name). Once you have done this, open up Intellij IDEA, close any open projects and click 'import project' (See Image 1). Navigate to the folder you extracted the MDK to, and select the 'build.gradle' file inside there. Then select 'Create directories for empty content roots automatically' and unselect 'Create separate module per source set' (See Image 2).

Setting up the Workspace

Now you have created the project, you need to setup the workspace. This requires changing the build.gradle file a little bit, and running 2 gradle tasks.

Setting up the build.gradle file

When setting up the build.gralde file, it is best to make a gradle config, this will make it easier for you to change version numbers. So open your 'gradle.properties' file and make it some something along the lines of...

Template
org.gradle.jvmargs=-Xmx3G

mc_version=[mc version]
forge_version=[forge version]
forge_mappings=[mcp version]

mod_name=[mod name]
mod_version=[mod version]
Example
org.gradle.jvmargs=-Xmx3G

mc_version=1.12.2
forge_version=latest
forge_mappings=20180618

mod_name=elementalgems
mod_version=1.0.0
